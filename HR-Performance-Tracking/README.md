HR departments often have to manually calculate scores to see who is meeting performance standards and who isn't. This project automates that burden.

Project Title: HR Performance Tracking & Automation System

The Problem: Manually evaluating employee performance across multiple test metrics (Safety, Philosophy, Financial, Drug test) is time-consuming and prone to human error.

The Solution: I developed an automated grading system in Excel that ingests raw test scores and immediately flags employees for review if they do not meet the company’s performance threshold.

Key Skills:

Implemented complex nested IF logic to automate decision-making.

Designed a dynamic dashboard that allows HR to quickly filter for "At-Risk" employees.

"Automated performance scoring, significantly reducing manual data entry and calculation time while eliminating human error in HR reporting."
