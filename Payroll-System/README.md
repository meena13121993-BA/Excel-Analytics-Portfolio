Automated Payroll & Overtime Calculation System
The Problem: Tracking variable hourly wages, overtime hours, and bonus structures across multiple weeks can lead to significant discrepancies.

The Solution: This project serves as a comprehensive payroll processing tool. It calculates weekly gross pay and overtime bonuses based on logged hours, providing a clean, monthly consolidated view.

Key Skills:

Advanced Data Aggregation (Summing hours and earnings across multiple time periods).

Handling Time-Series Data in Excel.

Building scalable formulas that calculate overtime premiums automatically once a 40-hour threshold is crossed.
